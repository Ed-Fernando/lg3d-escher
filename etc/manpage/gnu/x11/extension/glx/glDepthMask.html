- enable or disable writing into the depth buffer

C SPECIFICATION

PARAMETERS

       _param1  Specifies whether the depth buffer is enabled for
                writing.  If _param1 is  GL_FALSE,  depth  buffer
                writing  is  disabled.  Otherwise, it is enabled.
                Initially, depth buffer writing is enabled.

DESCRIPTION

       glDepthMask specifies whether the depth buffer is  enabled
       for writing.  If _param1 is GL_FALSE, depth buffer writing
       is disabled.  Otherwise, it is enabled.  Initially,  depth
       buffer writing is enabled.

ERRORS

       GL_INVALID_OPERATION  is  generated if glDepthMask is exe-
       cuted between the execution of glBegin and the correspond-
       ing execution of glEnd.

ASSOCIATED GETS

       glGet with argument GL_DEPTH_WRITEMASK
