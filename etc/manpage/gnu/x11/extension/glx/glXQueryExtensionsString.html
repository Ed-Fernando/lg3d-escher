glXQueryExtensionsString - return list of supported exten-
       sions

C SPECIFICATION

       const char * glXQueryExtensionsString( Display *dpy,
                                              int screen )

PARAMETERS

       dpy     Specifies the connection to the X server.

       screen  Specifies the screen number.

DESCRIPTION

       glXQueryExtensionsString  returns a pointer  to  a  string
       describing  which GLX extensions are supported on the con-
       nection. The string  is  null-terminated  and  contains  a
       space-separated  list  of extension names.  (The extension
       names themselves never contain spaces.) If  there  are  no
       extensions to GLX, then the empty string is returned.

NOTES

       glXQueryExtensionsString is available only if the GLX ver-
       sion is 1.1 or greater.

       glXQueryExtensionsString only  returns  information  about
       GLX  extensions.  Call  glGetString  to  get  a list of GL
       extensions.
