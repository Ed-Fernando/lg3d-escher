- specify the clear value for the depth buffer

C SPECIFICATION

PARAMETERS

       _param1  Specifies  the  depth  value  used when the depth
                buffer is cleared. The initial value is 1.

DESCRIPTION

       glClearDepth specifies the depth value used by glClear  to
       clear  the depth buffer.  Values specified by glClearDepth
       are clamped to the range [0, 1].

ERRORS

       GL_INVALID_OPERATION is generated if glClearDepth is  exe-
       cuted between the execution of glBegin and the correspond-
       ing execution of glEnd.

ASSOCIATED GETS

       glGet with argument GL_DEPTH_CLEAR_VALUE
