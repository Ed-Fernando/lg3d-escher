- determine if a name corresponds to a display-list

C SPECIFICATION

PARAMETERS

       _param1  Specifies a potential display-list name.

DESCRIPTION

       glIsList  returns GL_TRUE if _param1 is the name of a dis-
       play list and returns GL_FALSE otherwise.

ERRORS

       GL_INVALID_OPERATION is generated if glIsList is  executed
       between  the  execution  of  glBegin and the corresponding
       execution of glEnd.
