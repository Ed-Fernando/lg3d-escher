- specify clear values for the accumulation buffer

C SPECIFICATION

PARAMETERS

       _param1, _param2, _param3, _param4
                                        Specify  the  red, green,
                                        blue,  and  alpha  values
                                        used  when  the accumula-
                                        tion buffer  is  cleared.
                                        The  initial  values  are
                                        all 0.

DESCRIPTION

       glClearAccum specifies the red,  green,  blue,  and  alpha
       values used by glClear to clear the accumulation buffer.

       Values  specified by glClearAccum are clamped to the range
       [-1, 1].

ERRORS

       GL_INVALID_OPERATION is generated if glClearAccum is  exe-
       cuted between the execution of glBegin and the correspond-
       ing execution of glEnd.

ASSOCIATED GETS

       glGet with argument GL_ACCUM_CLEAR_VALUE

SEE ALSO

       glAccum, glClear
