glXGetCurrentDisplay - get display for current context

C SPECIFICATION

       Display * glXGetCurrentDisplay( void )

DESCRIPTION

       glXGetCurrentDisplay  returns  the display for the current
       context. If no context is current, NULL is returned.

       glXGetCurrentDisplay returns client-side  information.  It
       does  not  make  a round-trip to the server, and therefore
       does not flush any pending events.

NOTES

       glXGetCurrentDisplay is only supported if the GLX  version
       is 1.2 or greater.
