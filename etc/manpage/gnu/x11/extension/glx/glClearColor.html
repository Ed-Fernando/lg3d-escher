- specify clear values for the color buffers

C SPECIFICATION

PARAMETERS

       _param1, _param2, _param3, _param4
                                        Specify  the  red, green,
                                        blue,  and  alpha  values
                                        used   when   the   color
                                        buffers are cleared.  The
                                        initial values are all 0.

DESCRIPTION

       glClearColor specifies the red,  green,  blue,  and  alpha
       values used by glClear to clear the color buffers.  Values
       specified by glClearColor are clamped to the range [0, 1].

ERRORS

       GL_INVALID_OPERATION  is generated if glClearColor is exe-
       cuted between the execution of glBegin and the correspond-
       ing execution of glEnd.

ASSOCIATED GETS

       glGet with argument GL_COLOR_CLEAR_VALUE
