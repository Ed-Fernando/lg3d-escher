XNoOp - No Operation

SYNTAX

       XNoOp(display)
             Display *display;

ARGUMENTS

       display   Specifies the connection to the X server.

DESCRIPTION

       The XNoOp function sends a NoOperation protocol request to
       the X server, thereby exercising the connection.
